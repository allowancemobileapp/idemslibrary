# **App Name**: Idem Legal

## Core Features:

- Profile Showcase: Display Professor Idem's biography, education, and academic roles.
- Expertise Overview: Outline areas of expertise with links to detail pages.
- Publication Listings: Showcase publications with filtering and download options.
- Speaking Engagements: List talks, seminars, and appearances.
- Contact Form: Enable users to send messages, which are then stored in a database using Supabase and forward the content of the form to SendGrid.
- Booking Widget: Embed a Calendly widget for booking consultations.

## Style Guidelines:

- Primary color: Deep navy (#0B2545) for an authoritative and scholarly feel.
- Background color: Off-white (#F8F9FB), providing a clean and professional backdrop.
- Accent color: Gold (#CFA23B) to highlight key information and calls to action.
- Headings: 'Playfair', a modern serif font, lending elegance to headlines; Note: currently only Google Fonts are supported.
- Body: 'PT Sans', a sans-serif font providing excellent readability for body text; Note: currently only Google Fonts are supported.
- Law-related icons (books, scales) with a clean, modern design.
- Responsive design adapting to desktop, tablet, and mobile devices.
- Subtle animations such as card hover effects to add interactivity.
